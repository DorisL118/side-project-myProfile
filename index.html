SKILLS

前端語言

融合 JavaScript 的動態設計、HTML 標籤的運用、以及 CSS 屬性的優化，我能夠打造出引人入勝的網頁體驗。 同時，我熟悉 RWD 的原理與佈局配置，能夠確保網站在各種設備上都呈現出最佳效果。另外，我還精通 SCSS，能夠提升開發效率並保持結構清晰。 熟練運用 Bootstrap 5 和 Material UI 等工具來加速開發過程，同時確保網站具有現代感和優雅的外觀。

前端框架

透過 React 框架進行開發，我專注於組件（component）的功能規劃，以建立可重用且模組化的元素。 從而實現更加靈活和可擴展的應用程序架構。通過組件化的開發方式，我能夠提高代碼的重用性、可維護性和可擴展性，同時保持應用程序的整體結構清晰明了。

版本控制

對於我來說，Git 是開發過程中不可或缺的工具，我非常重視並深刻理解它的重要性。 Git 提供了強大的版本控制功能，能夠跟蹤代碼的每一次修改、變更和歷史記錄，這對於個人和團隊的開發工作都至關重要。

UIUX

我能夠將設計和技術有機地結合起來，以為用戶提供出色的網頁和網頁體驗。 不僅能夠設計出具有吸引力和易用性的界面，還能夠利用技術手段實現這些設計，確保網站或應用程序在各種設備上都能夠呈現出最佳效果。 同時，我還能夠促進團隊間的有效溝通和協作，與設計師、產品經理和其他開發人員合作，確保設計理念能夠得到有效實現，並在開發過程中保持透明和協調。
