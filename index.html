我是劉馨璘 Doris Liu，一位擁有不同角色技能的前端工程師。
從行銷公關的角色開始，我為完成當時負責的企業官網，逐步理解 UIUX 設計，並深入探討使用者體驗。通過不斷的學習與進修，我最終帶著滿滿的熱忱，決心踏入前端設計領域。
在這段過程中，我接觸了多個不同領域，這些經歷成為我成長的寶貴養分。不論是以行銷的角度策劃 SEO 優化策略，或是使用 Google Analytics 進行流量追蹤與分析；以公關的角度強化品牌亮點曝光；還是以設計的角度思考每個介面的使用體驗，如何讓功能互動順暢簡單且具美感，這些都豐富了我的視野和能力。在前端設計開發中，我致力於將團隊的設計完美呈現給使用者。擁有不同的職位經驗，使我在多方溝通時能更好地理解不同團隊的需求，提升溝通效率，加速前端開發進程。
這些多元的經歷，讓我在專案中能夠兼顧各個層面的需求，創造出更具價值和吸引力的作品。未來，我將持續學習，保持對前端設計的熱情，不斷推出更多優秀的作品，與團隊共同實現創新和卓越。

前端語言

融合 React 和 JavaScript 的動態設計、HTML 標籤的運用、以及 CSS 和 SCSS 屬性的優化，我能夠打造出最棒的網頁體驗。 同時，我熟悉 RWD 的原理與佈局配置，確保網站在各種設備上都呈現最佳效果。我熟練使用 Bootstrap 和 jQuery，能提升開發效率並保持結構清晰，確保網站具有現代感和優雅外觀。

前端框架

透過 React 框架進行開發，我專注於組件（component）的功能規劃，以建立可重用且模組化的元素，實現更加靈活和可擴展的應用程序架構。通過組件化的開發方式，我能夠提高代碼的重用性、可維護性和可擴展性，同時保持應用程序的整體結構清晰明了。使用 jQuery 進一步提高前端開發效率和互動性。

版本控制

Git 是開發過程中不可或缺的工具，我非常重視並深刻理解它的重要性。使用版本控制蹤代碼的每一次修改、變更和歷史記錄，這對於個人和團隊的開發工作都至關重要。除了前端技術，我也熟悉後端開發，尤其是使用 PHP 和 SQL 來構建和管理數據庫，確保前後端協同工作，提供完整的網站和應用程序解決方案。

POCA 跑咖

Website link ➔

市面上普遍關於咖啡廳的資訊都較零散，咖啡廳的文章、菜單以及詳細店家資訊往往需要在不同的網站做查詢，而 POCA 希望整合這些痛點。
除了讓訪問者可以依造喜好的類別找尋咖啡廳，也能觀看文章深入了解咖啡廳的魅力，以及詳細店家資訊。 同時致力於推廣公平貿易咖啡，為咖啡小農提供販售平台。 我們希望通過這個平台，讓更多人了解公平貿易的理念，並享受高品質的咖啡。

功能：
咖啡廳文章：分享有關咖啡文化和咖啡廳故事的文章，讓用戶依期望類別快速搜尋且了解咖啡廳。
購買咖啡：免登入會員，用戶可以輕鬆地選購他們喜愛的公平貿易咖啡，支持公平貿易運動。

負責項目：我負責網站的前端開發，實現文章快速篩選和購物車功能，確保用戶在每一步操作中都能享受到順暢和愉快的體驗。

企業品牌官網

旨在展示企業的核心價值和品牌理念，讓訪問者感受到企業的使命，快速聯繫我們，找到最適合的合作方式。 企業品牌網站是我進入前端領域的重要里程碑。在設計網站和界面的過程中，我請教了工程師同事許多程式相關的問題，並得到了他們的幫助和指導。在不斷的學習與積累下，我對前端開發產生了濃厚的興趣，最終走上了這條職業道路。

功能：
品牌理念：設計了一個優雅的頁面，展示企業的核心價值和品牌故事，讓用戶一目了然。
亮點展示：通過精心設計的圖文並茂的頁面，突出企業的主要業務和成功案例，吸引訪問者的注意。
解決方案：詳細介紹了企業提供的各種解決方案，幫助訪問者找到適合他們需求的服務。
連繫合作：提供簡單明了的聯繫方式和合作選項，方便潛在客戶與企業建立聯繫。

負責項目：我參與了前端設計和開發，致力於創建響應式佈局和動態內容展示，讓用戶在任何設備上都能享受到一致的視覺和使用體驗。

企業 Photo Gallery

這是一個充滿情感和故事的合作展示平台。我們通過深入且溫暖的文章，介紹企業合作的成功案例，讓用戶感受到合作背後的努力和成果。

在這個專案中，我負責UIUX設計規劃，以及基本前端開發，確保網站所想傳達的理念和用戶體驗達到最佳效果。 而我除了以行銷公關的身分，採訪合作品牌並撰寫新聞稿外，同時也負責 Photo Gallery 的視覺與功能設計、以及前端開發執行。在每一篇採訪中，皆特別融入了合作品牌特有的亮點，例如「 宜蘭行健有機村唯一民宿」，與品牌方的 Logo 和 CI 色，一點點的小巧思，除了讓網頁更有溫度，同時也更能傳達理念，創造更多的合作可能。

功能：
合作案例文章：每篇文章都精心撰寫，詳細介紹了合作背景、過程和成果，讓訪問者更深入地了解合作的細節。
搜索和篩選：設計了強大的搜索和篩選功能，讓用戶可以輕鬆找到感興趣的合作案例。
圖片和視頻展示：每個案例都附有相關的圖片和視頻，增強文章的吸引力和說服力。

負責項目：我負責UIUX設計規劃，以及基本前端開發，確保網站所想傳達的理念和用戶體驗達到最佳效果。
